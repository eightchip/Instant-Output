## 各機能のコスト計算

### 1. 再翻訳（Whisper + ChatGPT）
- **Whisper**: $0.006/分（平均30秒 = 0.5分として $0.003）
- **ChatGPT**: 入力約50トークン + 出力約100トークン = 150トークン
  - コスト: (50 × $0.15 + 100 × $0.60) / 1,000,000 = **$0.0000675/回**
- **合計**: 約**$0.003/回**

### 2. 文章改善
- **入力**: 約100トークン（英語文 + プロンプト）
- **出力**: 約150トークン（改善された文）
- **コスト**: (100 × $0.15 + 150 × $0.60) / 1,000,000 = **$0.000105/回**

### 3. 語彙抽出
- **入力**: 約150トークン（英語文 + プロンプト）
- **出力**: 約100トークン（JSON形式）
- **コスト**: (150 × $0.15 + 100 × $0.60) / 1,000,000 = **$0.0000825/回**

### 4. 品質チェック
- **入力**: 約200トークン（英語文 + 日本語訳 + プロンプト）
- **出力**: 約100トークン（JSON形式）
- **コスト**: (200 × $0.15 + 100 × $0.60) / 1,000,000 = **$0.00009/回**

### 5. 学習進捗分析
- **入力**: 約500トークン（統計データ + プロンプト）
- **出力**: 約500トークン（分析結果）
- **コスト**: (500 × $0.15 + 500 × $0.60) / 1,000,000 = **$0.000375/回**

### 6. OpenAI TTS
- **$15/100万文字**
- **1文字あたり**: $0.000015

### 7. OCR（画像認識）
- **入力**: 約500トークン（プロンプト）
- **画像**: 高解像度で約$0.01-0.02/回（Vision APIの料金）
- **出力**: 約500トークン（抽出テキスト）
- **コスト**: 約**$0.015/回**（画像処理コストが大きい）

## リスク要因

### 1. 為替変動
- 1ドル=150円 → 160円に変動した場合
- コストが約7%増加
- **対策**: 為替変動リスクを考慮してマージンを確保

### 2. OpenAI API価格改定
- API価格が上昇した場合
- **対策**: 価格改定時にプラン制限を調整

### 3. 異常使用
- 1ユーザーが大量に使用した場合
- **対策**: 1日の使用量上限を設定
